File: docs/getting-started/SETUP_GUIDE.md

1. Requirements â€‹

  • Node.js 18+: Download
  • Java 17+ (for Lavalink): Download
  • MongoDB: MongoDB Atlas (free) or local install
  • Discord Account: With server admin permissions

2. Create Discord Application â€‹

  • Go to Discord Developer Portal
  • Create a new application and bot user
  • Copy your bot token (keep it private!)
  • Enable required intents and permissions

3. Configure OAuth2 & Permissions â€‹

  • Use the OAuth2 URL Generator to select:
    • bot, applications.commands
    • Permissions: Send Messages, Use Slash Commands, Connect, Speak, Manage Roles, etc.

4. Clone & Install â€‹

  • Clone the repository
  • Install dependencies:
    bash
    npm install
  • Set up your .env file with your bot token and MongoDB URI

5. Start the Bot â€‹

  • Start the bot:
    bash
    npm start
  • (Optional) Set up Lavalink for music features

6. Troubleshooting â€‹

  • See the FAQ for common issues
